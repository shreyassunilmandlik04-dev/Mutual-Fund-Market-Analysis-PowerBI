# Mutual Fund Market Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes Mutual Fund industry performance from 2019 to 2025. The dashboard provides insights into Assets Under Management (AUM), investor participation, fund mobilization, redemptions, and net inflows.

## Tools Used
- Power BI
- Microsoft Excel
- CSV Dataset

## Dataset Information
The dataset contains:
- Date
- Scheme Type
- Scheme Category
- Number of Schemes
- Number of Folios
- Funds Mobilized
- Redemption Amount
- Net Inflow/Outflow
- Assets Under Management (AUM)

## Key Performance Indicators (KPIs)
- Total AUM
- Total Folios
- Total Schemes
- Funds Mobilized
- Redemption Amount
- Net Inflows

## Dashboard Insights
- Growth trends in Mutual Fund investments
- Investor participation analysis
- Fund flow analysis
- Category-wise performance comparison
- Industry growth trends from 2019 to 2025
